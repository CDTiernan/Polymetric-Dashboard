Defining the namespace, sensor, and item attributes allows the State Ion to make REST API calls to get the sensor's state.

{{namespace}}
{{sensor}}
{{item}}

Defining the state attribute forces the State Icon to display the defined state, it will not make REST API calls.

{{state.label}}
Apply

HTML

  
    <smp-state-icon
      namespace = "{{scStateIcon.namespace}}"
      sensor = "{{scStateIcon.sensor}}"
      item = "{{scStateIcon.item}}"
    > </smp-state-icon>
  

  
    <smp-state-icon
      state = "{{scStateIcon.state}}"
    > </smp-state-icon>
