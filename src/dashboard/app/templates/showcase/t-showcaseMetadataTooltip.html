Hover to show metadata

Place the Metadata Tooltip within any html element. Now when a user hovers over its parent element the Metadata Tooltip will become visible.

{{namespace}}
{{sensor}}
{{item}}
Apply

HTML

  
    <div>
      <p>Hover to show metadata</p>

      <smp-metadata
        namespace = "{{scMetadata.namespace}}"
        sensor = "{{scMetadata.sensor}}"
        item = "{{scMetadata.item}}"
      > </smp-metadata>
    </div>
