Tab

HTML Template

Description

Each tab requires a template to define what it will display. Templates are placed in the /app/templates/tabs directory, and are an html file. Use of Angluar Material, NVD3 charts, and all the available visualization tools are encouraged during development of a tab.

Example

app/tempates/tabs/t-exampleView.html

  
    <div layout="row" layout-wrap>
      <div>
        <h2 class="md-headline smpHeader">Example View</h2>
      </div>

      <div layout="column" flex="100">
        <h4 class="md-title chartTitle">CPU Usage</h4>
        <smp-nvd3-chart
          id = "showcaseTab-CPU"
          type = "lineChart"
          namespace = "%SYS"
          sensor = "CPUusage"
          item = "-"
          units = "%"
          height = "144"
        > </smp-nvd3-chart>
      </div>
    </div>

JavaScript Directive

Description

In order for angular to know what template to inject into the DOM each tab template needs to be referenced by an Angluar Directive. Directives are place in the /app/directives/tabs directory, and are JavaScript files defining where the tab template is located as well as some AngluarJS parameters for the tab.

Example

app/directives/tabs/t-exampleView.js

  
    (function() {
      'use strict';

      angular.module('Dashboard')
        .directive('exampleView, function() {
          return {
            restrict: 'E',
            templateUrl: 'app/templates/tabs/t-exampleView.html',
            controller: 'exampleCtrl as example',
          };
        });
    })();

Adding To System Dashboard

Description

Tabs are dynamically added to the application using AngularJS. Thus to add a new tab, the directive needs to be placed in the tabs array held in the content controller, which can be found in /app/controllers/core/c-content.js. To add a new tab simply create a new object in the tabs array defining an id for the tab, a title, the html reference to the directive, and the width the tab should fill on the screen.

Example

  
    $scope.tabs = [
      {
        id: 'summary', title: 'Summary', width: 80,
        content: '<summary-view ng-if="$index === data.selectedIndex"></summary-view>'
      },
      {
        id: 'example', title: 'Example', width: 80,
        content: '<example-view ng-if="$index === data.selectedIndex"></example-view>'
      }
    ];
